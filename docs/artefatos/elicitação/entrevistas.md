# Entrevistas

## Introdução

A entrevista é uma das primeiras técnicas usadas na elicitação de requisitos, ela estabelece um dialogo entre a equipe de desenvolvimento e os prováveis usuários com a finalidade de entender melhor as necessidades do projeto.

## Metodologia

Nossa metodologia para as entrevistas foi baseada na elaboração de perguntas com foco em entender os requisitos usados para o desenvolvimento do Dioarti. Para a entrevista, usamos as próprias personas como entrevistados e fizemos as seguintes perguntas:

1. Qual funcionalidade você achou melhor no aplicativo Doarti?
2. Você precisou, em média, de quanto tempo para executar uma ação no aplicativo?
3. Você já fez alguma doação ou anunciou seu pedindo no aplicativo?
4. Na sua opinião, alguma coisa poderia ser diferente?
